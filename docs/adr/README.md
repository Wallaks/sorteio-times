# Architecture Decision Records (ADR)

## Formato

Cada ADR é um arquivo: `NNN-titulo-curto.md` com:

- **Status:** proposto / aceito / substituído
- **Contexto**
- **Decisão**
- **Consequências**

## Índice

| ADR | Título | Status |
|-----|--------|--------|
| — | *(ex.: escolha Vite + TS)* | *(a criar quando a stack for fixada)* |

## Quando criar um ADR

- Escolha de framework, formato de persistência, ou política de corte (local vs servidor).
- Mudança que afeta contrato de sorteio ou compatibilidade com listas antigas do grupo.
